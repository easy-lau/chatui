# ChatUI v1.7.3

Released on July 20, 2026.

## Fixes

- Follow-up source markers no longer change the alignment of the user's message body after intent recognition completes.
- User message text keeps its normal leading alignment while the message bubble and attachment layout remain positioned on the user side.
- Bumped the message stylesheet cache key so deployed browsers receive the corrected layout immediately.

## Verification

- Added a regression test that prevents quote or follow-up metadata from right-aligning user message text.
- Verified the full clarification flow in Microsoft Edge against a simulated OpenAI-compatible upstream: initial clarification, follow-up classification, marker insertion, final response, and task completion.
- Confirmed the composer returns from the stop/running state to the enabled send icon after both clarification completion and final task completion.
- `npm run check` passes with 362 tests.
