# ChatUI v1.7.7

Released on July 20, 2026.

## Improvements

- Added height-aware welcome-page layouts for low-resolution desktop displays.
- Kept the welcome content visually centered whenever it fits while reserving safe space above the fixed composer.
- Added an extra-dense fallback for extremely short desktop viewports.
- Added sufficient mobile scroll clearance so the usage notice can be moved fully above the composer.

## Fixes

- Prevented the welcome page from being clipped or covered by the composer on common 1366x768, 1024x600, and 800x600 displays.
- Refreshed the application and static bundle cache versions so browsers do not reuse an older corrupted welcome script that renders Chinese text as question marks.
- Preserved the welcome copy as ASCII-safe JavaScript Unicode escapes to avoid encoding loss on Windows tooling.

## Verification

- Verified the welcome layout in Microsoft Edge at 1366x768, 1280x720, 1024x768, 1024x600, 800x600, 761x500, 390x844, and 360x640.
- Confirmed the rendered welcome title matches the expected Chinese product name and contains no replacement question-mark sequences.
- `npm run check` passes with 371 tests.
