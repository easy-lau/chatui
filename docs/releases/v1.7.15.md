# ChatUI v1.7.15

Released on July 22, 2026.

## Fixes

- Removed the local image-composition keyword fallback that could overwrite a valid `task_contract.v3` response with an unrelated image clarification.
- Routing now executes only the strictly validated model contract; image selection, ambiguity, and clarification remain part of that contract instead of being inferred from local keywords or candidate scoring.
- Removed the residual no-op pending-task local fallback from both submission entry paths; a failed or invalid continuation decision is now always a new task.
- Bumped the router, submission workflow, and root-entry asset versions so browsers fetch the corrected logic after deployment.

## Verification

- Added regression coverage for a standalone business request containing “多对多” and “自动合并行展示/分组” while prior image candidates exist.
- Preserved model-declared multi-image routing coverage and verified the end-to-end multipart image request.
