# ChatUI v1.7.16

Released on July 22, 2026.

## Improvements

- Fenced Markdown code now appears as a live code block as soon as the opening fence is complete, rather than exposing raw fence markers during streaming.
- Streamed code is appended incrementally and syntax-highlighted on a short throttle; the completed response continues through the normal Markdown rendering and enhancement flow.
- Bundled Highlight.js 11.11.1 with expanded language support, including Java, and refreshed the light code surface for clearer contrast.

## Verification

- Added regression coverage for live fenced-code rendering, incremental appends, highlight throttling, timer cleanup, safe highlighted markup, and Java syntax support.
- `npm run check` passes with 395 automated tests.
