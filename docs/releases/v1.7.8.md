# ChatUI v1.7.8

Released on July 20, 2026.

## Improvements

- Replaced the welcome-page center card with a dedicated animated globe for a cleaner, more focused first-run experience.
- Added layered atmosphere, ocean, grid, land, glint, aura, tilt, and floating effects to give the globe more depth.
- Tuned the globe size for compact desktop layouts while preserving the surrounding capability orbit.
- Disabled the new globe animations when the operating system requests reduced motion.

## Fixes

- Removed the redundant `AI` and `READY` labels from the center of the welcome illustration.
- Refreshed the application, theme, and static-bundle cache versions so browsers receive the updated welcome assets immediately.

## Verification

- Added regression coverage for the globe-only markup, responsive styling, animation hooks, reduced-motion behavior, and cache versions.
- `npm run check` passes.
