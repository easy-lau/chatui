# ChatUI v1.7.14

Released on July 22, 2026.

## Fixes

- Prevented an old image clarification from being applied to a complete, unrelated new request.
- Made pending-task continuation model-driven and contract-validated: only a complete, high-confidence `pending_continuation.v1` result may resume a prior task.
- Treat malformed, incompatible, timed-out, or unavailable continuation classification as a new request instead of inheriting prior task context.

## Verification

- Added regression coverage that rejects `task_contract.v3` as a pending-task continuation decision and accepts only the strict continuation contract.
- `npm run check` passes with 392 automated tests.
